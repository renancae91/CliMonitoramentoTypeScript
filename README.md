# cli-monitor

1. Instale as dependências e faça o build do projeto:
    ```bash
      $ npm run install-build
    ```

2. Instale o porjeto globalmente para usar a CLI:
    ```bash
      $ npm install -g
    ```
    ou
    ```bash
      $ npm link
    ```

3. Execute a CLI no seu terminal:
    ```bash
      $ startCli 
    ```# CliMonitoramentoTypeScript
